простая иммитацию интернет-магазина, карточки, хедер, футер, детализация карточек, личный кабинет, корзина и избранное.

 поиск по товару

Проект включает в себя:
- vite
- react
- type script
- module css
- router через createBrowserRouter
- layout (каркас с outlet)
- localstorage
- авторизация через отдельнй хук
- контроль над задачами через jira
- данные через data (позже через api)
- zustand
- ci/cd gitHUb Actions

- состояние через zustand:
    Каждый пользователь хранит свои данные отдельно (usersData[email]).
    Cart и Favorites не теряются при logout.
    Persist хранит всё состояние автоматически.
    Типы полностью безопасные — TypeScript контролирует cart, favorites и user.
    Можно легко расширять функционал, например добавлять настройки пользователя или историю заказов, отзывы навигацию по категориям, поиск по товару.
