простая иммитацию интернет-магазина, карточки, хедер, футер, детализация карточек, поиск по товару, личный кабинет с корзина и избранным.

Проект включает в себя:
- vite
- react
- ts
- router через createBrowserRouter и layout
- localstorage
- авторизация через отдельнй хук
- контроль над задачами через jira
- данные через data (позже через api)

- состояние через zustand
    Каждый пользователь хранит свои данные отдельно (usersData[email]).
    Cart и Favorites не теряются при logout.
    Persist хранит всё состояние автоматически.
    Типы полностью безопасные — TypeScript контролирует cart, favorites и user.
    Можно легко расширять функционал, например добавлять настройки пользователя или историю заказов.
